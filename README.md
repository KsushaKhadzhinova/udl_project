# UDL Editor - Universal Diagram Language

**CASE-средство для визуального моделирования.**

## Технологический стек:
- **Backend:** FastAPI (Python)
- **Database:** Neo4j (Graph DB)
- **Notation support:** IDEF0, Petri Nets, UML, DFD.

## Структура проекта:
- `/backend`: Логика парсинга UDL и API.
- `/docs`: Лабораторные работы и проектная документация.
