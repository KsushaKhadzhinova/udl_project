# Палитра UI для CASE-средства (Лаба №4)

Профессиональный инженерный интерфейс, адаптированный для темной/светлой тем и WCAG AA. Основа — правило 60-30-10.

## Темная тема (Industrial Dark)

| Цвет | HEX | Описание |
|------|-----|----------|
| Основной (60%) | `#1E1E1E` | Фон (VS Code стиль, без "гало") |
| Вторичный (30%) | `#252526` | Сайдбары, панели |
| Акцентный (10%) | `#007ACC` | Кнопки, фокус (контраст >4.5:1) |
| Границы | `#3C3C3C` | Разделители |

## Светлая тема (Industrial Light)

| Цвет | HEX | Описание |
|------|-----|----------|
| Основной (60%) | `#FFFFFF` | Фон |
| Вторичный (30%) | `#F3F3F3` | Сайдбары, панели |
| Акцентный (10%) | `#005FB8` | Кнопки, фокус |
| Границы | `#E5E5E5` | Разделители |

## Шрифты

- **UI**: `Inter` (читаемость 11-12px)
- **Код/UDL**: `Fira Code` (лигатуры →, ==)

## A11y (WCAG AA)

- Контраст акцентов >4.5:1
- `aria-label` для иконок
- `outline` для фокуса
- Скругление: `2px`

## CSS-переменные

```css
:root {
  --main-bg-dark: #1E1E1E;
  --secondary-bg-dark: #252526;
  --accent-dark: #007ACC;
  --border-dark: #3C3C3C;
  
  --main-bg-light: #FFFFFF;
  --secondary-bg-light: #F3F3F3;
  --accent-light: #005FB8;
  --border-light: #E5E5E5;
  
  --text-primary: #CCCCCC;
  --text-secondary: #808080;
  
  --font-ui: 'Inter', sans-serif;
  --font-code: 'Fira Code', monospace;
  
  --border-radius: 2px;
}
```
